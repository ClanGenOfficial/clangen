## Running from source
### Requirements
- python >= 3.8
- pygame >= 2.1.3.dev8
- pygame_gui >= 0.6.8

### Setting up
Run `pip install -r requirements.txt` to install dependencies. 

### Running game
Run `python3 main.py`. If python3 does not exist try running `python main.py` instead.
